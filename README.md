## 📋 Requirements

- Python 3.8+
- Linux-based OS (Kali Linux recommended)
- Root privileges (for some tools)

## 🚀 Installation

### Method 1: Git Clone

```bash
git clone https://github.com/your-username/hackingtool.git
cd hackingtool
chmod +x install.sh
sudo bash install.sh
```

### Method 2: Docker

```bash
docker build -t hackingtool .
docker run -it hackingtool
```
